CSS elements

THE FONT-FAMILY PROPERTY
  • Configures font typeface. A web browser displays text using the fonts that have been installed on the userâ€™s computer. Times New Roman is the default font for most browsers.
  • The Verdana, Tahoma, and Georgia font typefaces were specifically designed to display well on computer monitors.
  • Common Practice is to use a serif font (such as Georgia or Times New Roman) for headings and a sans-serif font (such as Verdana or Arial) for detailed context
THE FONT-SIZE PROPERTY
  • Sets the size of the font. Commonly used categories of font-size values are text value, pixel unit (px), point unit (pt), em unit (em), percentage value (%) and many more.
  • A px value is static. This is an OS-independent and cross-browser way of literally telling the browsers to render the letters at exactly the number of pixels in height that you specified.
